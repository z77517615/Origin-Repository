# [台北一日遊]旅遊電商專案
### 探索每個角落，體驗城市的深度旅遊行程

<br/>

## 使用技術
* 原生HTML+CSS完成RWD網頁。
* RESTful API架構實踐專案功能。
* Python flask建立Application server。
* MySQL關聯式資料庫儲存會員資料及訂單。
* 結合TapPay第三方金流服務，開發購物車系統。
* 專案部署於AWS EC2。

<br/>
